# 架构决策记录 (ADR)

本目录记录开心农场项目中的重要架构决策。

## 什么是 ADR

架构决策记录（Architecture Decision Record）是一种轻量级文档，用于记录项目中重要的架构决策、其背景、后果和替代方案。

## ADR 格式

每个 ADR 文件命名格式：`NNNN-决策简述.md`

使用以下模板：

```markdown
# ADR-NNNN: 决策标题

**状态**：[提议 | 已接受 | 已废弃 | 已取代]
**日期**：YYYY-MM-DD
**决策者**：[姓名/角色]

## 背景
描述需要做出决策的上下文和问题。

## 决策
描述我们做出的决策。

## 后果
### 正面影响
### 负面影响
### 风险

## 考虑的替代方案
1. 替代方案A - 描述及为何不可行
2. 替代方案B - 描述及为何不可行

## 参考资料
- [相关链接或文档]
```
