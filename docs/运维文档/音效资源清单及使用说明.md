## 🎛️ 音效控制面板功能

### 主音量控制
- **范围**：0% - 100%
- **功能**：控制所有音效的整体音量
- **默认值**：70%

### 音效音量控制
- **范围**：0% - 100%
- **功能**：独立控制游戏音效的音量
- **默认值**：70%

### 音乐音量控制
- **范围**：0% - 100%
- **功能**：独立控制背景音乐的音量
- **默认值**：50%

### 背景音乐开关
- **功能**：开启/关闭背景音乐
- **默认状态**：关闭

### 全部静音开关
- **功能**：一键静音/取消静音所有音效
- **快捷键**：点击右上角喇叭图标

### 预设音量
- **全开**：主音量100%、音效100%、音乐80%
- **半开**：主音量50%、音效50%、音乐30%
- **关闭**：全部静音

### 测试按钮
- **测试点击**：播放点击音效
- **测试成功**：播放成功音效
- **测试金币**：播放金币音效

## 💾 音效设置持久化

### 存储位置
- **存储介质**：浏览器 `localStorage`
- **存储键名**：`happyFarm_soundSettings`

### 存储内容
```json
{
  "masterVolume": 0.7,
  "soundVolume": 0.7,
  "musicVolume": 0.5,
  "isMuted": false
}
```

### 特性
- ✅ 自动加载上次保存的设置
- ✅ 实时保存设置变更
- ✅ 刷新页面后设置保持不变
- ✅ 跨会话持久化

## 🚀 开发者使用指南

### 1. 音效管理器 (soundManager.js)

#### 初始化
```javascript
import soundManager from './services/soundManager';

// 自动初始化（首次播放时）
```

#### 播放音效
```javascript
// 播放指定音效
soundManager.play('plant');
soundManager.play('harvest');
soundManager.play('coin');
```

#### 音量控制
```javascript
// 设置主音量 (0-1)
soundManager.setMasterVolume(0.8);

// 设置音效音量 (0-1)
soundManager.setSoundVolume(0.6);

// 设置音乐音量 (0-1)
soundManager.setMusicVolume(0.4);

// 静音
soundManager.setMuted(true);

// 切换静音
const isMuted = soundManager.toggleMute();
```

#### 背景音乐
```javascript
// 播放背景音乐
soundManager.playBackgroundMusic();

// 停止背景音乐
soundManager.stopBackgroundMusic();
```

#### 获取音效列表
```javascript
const soundList = soundManager.getSoundList();
console.log(soundList);
// 输出:
// {
//   count: 14,
//   names: ['plant', 'harvest', 'unlock', ...],
//   hasBackgroundMusic: true
// }
```

### 2. 音效设置组件 (SoundSettings.vue)

#### 引入组件
```vue
<script setup>
import SoundSettings from './components/SoundSettings.vue';
</script>

<template>
  <div>
    <SoundSettings />
  </div>
</template>
```

## 🎨 音效与视觉同步

### 同步策略
- 音效在动画开始时触发
- 动画时长与音效时长匹配
- 关键帧与音效节奏同步

### 示例场景

2. **收获**：点击 → 播放收获音效 → 显示收获动画 → 播放金币音效
3. **解锁**：确认 → 播放解锁音效 → 显示解锁动画

## 🔧 音量平衡

### 音量层级
- **背景音乐**：0.08 × 音乐音量 × 主音量（轻柔）
- **游戏音效**：0.15-0.4 × 音效音量 × 主音量（适中）
- **UI反馈音**：0.1-0.3 × 音效音量 × 主音量（轻柔）

### 设计原则
- 背景音乐不干扰游戏音效
- 重要音效（收获、升级）更突出
- UI反馈音轻柔不突兀
