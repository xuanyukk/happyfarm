## 四、批量生成脚本参考

### 4.1 使用 Qwen 工具批量生成

```bash
# 进入生成工具目录
cd frontend/scripts/generate-icons/

# 生成所有缺失的UI元素（粒子、头像框、装饰元素）
node generateIcons.cjs --type ui --output missing_ui

# 生成所有缺失的空状态和背景
node generateIcons.cjs --type empty --output missing_empty
```

### 4.2 使用 Kolors 工具生成背景图

```bash
# 生成农场背景图（需要大尺寸，Kolors支持256x256，背景图需拼接或放大）
node kolors-generate-crops.cjs --type background --output missing_bg
```

> 注：1920×1080px 背景图超出 Kolors 默认尺寸，建议：
> 1. 使用 FLUX 工具生成（支持更高分辨率）
> 2. 或使用 AI 图像放大工具（如 Upscayl）将 256×256 放大到目标尺寸

## 五、风格一致性检查清单

生成所有图片后，请对照以下清单确保风格统一：

- [ ] 所有图标使用**扁平化矢量风格**
- [ ] 所有图标有**深色轮廓描边**
- [ ] 光源统一为**顶部偏左**
- [ ] 阴影统一为**柔和投影**（非硬阴影）
- [ ] 背景统一为**透明**（PNG格式）
- [ ] 色彩饱和度**中等偏高**，确保在暗色土地上可见
- [ ] 所有文件命名符合**snake_case**规范
- [ ] 所有文件已转换为**PNG-24**格式
