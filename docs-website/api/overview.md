# API 概览

本文档提供开心农场项目的 API 概览。

## 主要 API 模块

- 用户认证与授权
- 玩家与农场管理
- 作物系统
- 商店与经济
- 数据仓库与 BI 分析
- 后台管理

## API 文档

完整的 API 文档可以通过 Swagger UI 访问：

| 部署方式 | 地址 |
|---------|------|
| Docker 环境 | http://localhost:3001/api-docs |
| 本地开发环境 | http://localhost:3001/api-docs |

## 技术栈

### 后端
- Node.js 18+
- Express 5.x
- PostgreSQL 13+
- Redis 7+

### 前端
- Vue.js 3.x
- Vite 5.x
- Vue Router 4.x
- Pinia 3.x
