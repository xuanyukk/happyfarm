# 介绍

欢迎使用开心农场！这是一个完整的农场模拟游戏项目。

## 项目概述

开心农场是一个完整的农场模拟游戏项目，玩家可以通过种植作物、解锁地块、提升品质来获得农场币，实现完整的游戏循环。
