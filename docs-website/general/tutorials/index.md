## 教程分类概览

视频教程按以下模块进行分类：

- **部署教程** — Docker 部署、本地环境搭建、生产环境配置
- **前端开发** — Vue 3 组件开发、状态管理、路由配置
- **后端开发** — Express 控制器、服务层、中间件
- **数据库管理** — PostgreSQL 表结构、SQL 初始化、数据迁移
- **游戏功能** — 农场管理、作物系统、商店系统、成就系统
- **监控运维** — PM2 进程管理、日志平台、性能监控
- **开发工具** — Vite 配置、Git 工作流、测试编写
