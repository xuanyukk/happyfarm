
# 多货币体系

## 概述

开心农场采用策略模式驱动的可扩展多货币体系架构，支持多种货币类型和灵活的格式化显示。系统当前支持**农场币**和**农场宝石币**两种货币，并预留了未来扩展更多货币的能力。

## 货币配置

货币配置存储在 `currency_config` 表中，支持：
- 货币代码（currency_code）
- 货币名称和符号
- 格式化规则（format_rules）
- 显示精度
- 是否激活（is_active）
- 货币上限：农场币 999亿，宝石币 999,999

## 货币类型说明

### 1. 农场币（主要货币）

| 属性 | 说明 |
|------|------|
| **货币代码** | `FARM_COIN` |
| **数据库字段** | `coin_num` |
| **描述** | 主要游戏货币，用于日常游戏消费 |
| **显示符号** | 💰 |
| **上限** | 99,999,999,999（999 亿） |
| **格式化显示** | 万/亿体系（例如：1.2 万、1.5 亿） |

### 2. 农场宝石币（高级货币）

| 属性 | 说明 |
|------|------|
| **货币代码** | `GEM` |
| **数据库字段** | `gem_num` |
| **描述** | 高级货币，用于购买特殊道具和参与高级活动 |
| **显示符号** | 💎 |
| **上限** | 999,999 |
| **格式化显示** | 标准数字格式（例如：1,234） |

## 获取途径

### 农场币获取途径

| 途径 | 说明 | 预计收益 |
|------|------|----------|
| **出售作物** | 将成熟的作物收获后出售 | 根据作物类型和品质，收益不等 |
| **完成任务** | 完成日常任务和活动任务 | 固定奖励 + 随机奖励 |
| **成就奖励** | 解锁新成就获得一次性奖励 | 一次性大量奖励 |
| **登录奖励** | 每日首次登录获得 | 每日递增（7 天为一个周期） |
| **活动参与** | 参与游戏活动获得 | 根据活动类型奖励不同 |

### 农场宝石币获取途径

| 途径 | 说明 | 预计收益 |
|------|------|----------|
| **活动奖励** | 参与特殊活动或完成特殊任务 | 少量奖励 |
| **成就奖励** | 解锁部分高级成就 | 一次性奖励 |
| **等级升级** | 农场等级或世界等级升级 | 升级礼包中包含 |

::: warning 注意
当前版本暂未开放货币兑换功能，敬请期待后续更新！

规划中的兑换功能将包括：

## 限制条件

### 余额上限

- 农场币：99,999,999,999（999 亿）
- 农场宝石币：999,999

::: warning 注意
当您的货币余额接近上限时，新获得的货币可能会被截断（不会超过上限）。建议及时消费或捐赠。
:::

### 单次交易限制

| 货币类型 | 单次最小交易 | 单次最大交易 |
|----------|-------------|-------------|
| 农场币 | 1 | 1,000,000,000 |
| 农场宝石币 | 1 | 10,000 |

### 其他限制

- 货币无法直接交易给其他玩家
- 货币无法通过邮件等方式赠送
- 部分特殊活动奖励有领取时间限制
- 离线期间可通过委托系统获得少量农场币收益

## 格式化显示说明

系统采用**万/亿体系**格式化显示农场币，提升大数字的可读性。

### 格式化规则

| 数值范围 | 显示格式 | 示例 |
|----------|----------|------|
| 0 - 9,999 | 原始数字 | 1,234 |
| 10,000 - 99,999,999 | "X万" | 1.2万、15万 |
| 100,000,000 - 99,999,999,999 | "X.X亿" | 1.2亿、15.5亿、999亿 |

### 格式化特点

- 使用截断法（而非四舍五入）避免精度问题
- 整数倍数时自动去除小数位（例如 100,000 → 10万，而非 10.0万）
- Hover 悬停时可查看完整数值

## 常见问题

### Q：如何查看我的货币余额？
**A：** 登录后在游戏主界面顶部即可查看当前货币余额，或在「个人中心」查看详细余额和流水记录。

### Q：出售作物能获得多少农场币？
**A：** 不同作物的售价不同，请在作物详情或商店界面查看具体售价。高品质作物售价更高。

### Q：农场宝石币可以通过游戏免费获取吗？
**A：** 可以！通过完成特殊成就、参与活动和升级等级均可获得少量农场宝石币。

### Q：货币达到上限后会怎样？
**A：** 当您的货币达到上限后，新获得的货币将被截断，无法继续累积。建议及时消费或使用。

### Q：可以查看历史货币流水吗？
**A：** 可以！在「个人中心」→「货币流水」中可查看所有收入和支出记录。

## 技术架构说明（仅供开发者参考）

多货币体系采用策略模式设计，主要类结构如下：

| 类名 | 说明 |
|------|------|
| `CurrencyFormatter` | 货币格式化策略基类 |
| `FarmCoinFormatter` | 农场币格式化策略 |
| `GemFormatter` | 农场宝石币格式化策略 |
| `CurrencyConfig` | 货币配置基类 |
| `FarmCoinConfig` | 农场币配置 |
| `GemConfig` | 农场宝石币配置 |
| `CurrencyConfigFactory` | 货币配置工厂 |
