# 贡献指南

我们欢迎任何形式的贡献！

## 贡献流程

1. Fork 本仓库
2. 创建特性分支（`git checkout -b feature/AmazingFeature`）
3. 提交你的更改（`git commit -m 'feat: Add some AmazingFeature'`）
4. 推送到分支（`git push origin feature/AmazingFeature`）
5. 开启一个 Pull Request

## 提交前检查清单

- [ ] 代码已通过 ESLint 检查
- [ ] 已添加必要的测试用例
- [ ] 所有测试已通过
- [ ] 文档已更新（如适用）
- [ ] 提交信息遵循语义化规范
