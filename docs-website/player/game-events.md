
# 游戏活动系统

## 概述

游戏活动系统是开心农场的重要组成部分，为玩家提供丰富的游戏内容和奖励。系统支持多种活动类型，包括限时活动、节日活动、主题活动等，并提供实时通知、进度推送和奖励发放功能。

---

## 活动类型

### 1. 日常活动

| 活动名称 | 说明 | 频率 |
|----------|------|------|
| **每日签到** | 每日登录领取签到奖励 | 每日 |
| **日常任务** | 完成指定任务获得奖励 | 每日刷新 |
| **丰收节** | 收获指定数量作物获得额外奖励 | 每日限时 |

### 2. 限时活动

| 活动名称 | 说明 | 持续时间 |
|----------|------|----------|
| **节日活动** | 春节、中秋等节日主题活动 | 节日期间 |
| **主题活动** | 春季种植、秋季收获等主题 | 1-2 周 |
| **挑战赛** | 限时挑战特定目标 | 1-3 天 |

### 3. 长期活动

| 活动名称 | 说明 | 持续时间 |
|----------|------|----------|
| **成就收集** | 解锁成就获得奖励 | 永久 |
| **排行榜赛季** | 赛季排行竞争 | 1 个月 |

---

## 参与条件

### 基础条件

- **账号等级**：玩家账号需达到 5 级（农场等级）
- **完整注册**：完成账号注册和新手引导
- **在线状态**：活动期间需保持在线（部分活动支持离线参与）

### 特定活动条件

| 活动类型 | 额外条件 |
|----------|----------|
| **节日活动** | 无额外条件（所有玩家可参与） |
| **主题活动** | 可能需要特定农场等级或解锁特定作物 |
| **挑战赛** | 可能需要报名或达到特定等级 |
| **排行榜活动** | 无额外条件（所有玩家自动参与） |

## 参与流程

### 1. 查看活动

1. 登录游戏
2. 点击主界面「活动」按钮或导航到活动页面
3. 查看当前进行中的活动和即将开始的活动
4. 点击活动卡片查看活动详情

### 2. 报名参与

1. 在活动详情页查看参与条件和规则
2. 如需报名，点击「立即报名」按钮
3. 确认报名信息后完成报名
4. 报名成功后会收到通知

### 3. 完成任务

1. 查看活动任务列表和目标要求
2. 在游戏中完成对应任务
3. 任务进度实时更新
4. 完成任务后可在活动页面领取奖励

### 4. 领取奖励

1. 在活动页面查看可领取的奖励
2. 点击「领取奖励」按钮
3. 奖励将自动发放到您的账户
4. 可在「个人中心」→「奖励记录」查看历史奖励

## 奖励机制

### 奖励类型

| 奖励类型 | 说明 |
|----------|------|
| **农场币** | 主要游戏货币 |
| **农场宝石币** | 高级货币 |
| **道具** | 加速、防虫等道具 |
| **种子** | 限定或高级种子 |
| **经验值** | 玩家和农场经验 |
| **成就** | 解锁特殊成就 |
| **徽章** | 活动专属纪念徽章 |

### 奖励发放方式

| 发放方式 | 说明 | 到账时间 |
|----------|------|----------|
| **即时发放** | 完成任务后立即领取 | 即时 |
| **活动结束发放** | 活动结束后统一发放 | 活动结束后 24 小时内 |
| **排行榜奖励** | 根据排名发放 | 赛季结束后 48 小时内 |

### 奖励领取规则

- 奖励需在有效期内领取，过期失效
- 部分奖励有领取等级限制
- 同一活动奖励只能领取一次
- 奖励发放后会有系统通知

## 活动进度查看

### 个人进度

在活动详情页可查看：
- 当前已完成的任务数
- 个人进度百分比
- 距离下一奖励的进度
- 预计完成时间

### 排行榜

在排行榜页面可查看：
- 当前排名
- 前 100 名玩家信息
- 与前一名的差距
- 实时排名更新

### 通知推送

系统会通过以下方式推送活动信息：
- 游戏内通知中心
- WebSocket 实时推送
- 活动开始/结束提醒
- 任务完成提醒
- 奖励到账提醒

## 常见问题

### Q：如何查看当前有哪些活动？
**A：** 点击主界面「活动」按钮，或导航到活动页面，即可查看所有进行中和即将开始的活动。

### Q：活动错过时间可以补参与吗？
**A：** 部分活动支持，具体请查看活动规则说明。一般情况下，已结束的活动无法补参与。

### Q：奖励发放延迟怎么办？
**A：** 若奖励未按时到账，请等待 24 小时，如仍未到账可联系客服提交工单。

### Q：可以同时参与多个活动吗？
**A：** 可以！您可以同时参与多个活动，任务进度互不冲突。

### Q：活动任务太难了怎么办？
**A：** 活动任务分为不同难度等级，您可以选择适合自己的难度参与。部分活动支持组队参与。

### Q：如何取消活动报名？
**A：** 已报名的活动一般不支持取消，请谨慎报名。部分活动可在开始前取消，请查看活动详情页。

### Q：活动奖励可以交易或赠送吗？
**A：** 活动奖励不可交易或赠送，仅可在当前账号使用。

### Q：排行榜多久更新一次？
**A：** 排行榜一般每小时更新一次，部分活动支持实时更新。

## 活动规则

### 基本规则

1. **公平参与**：严禁使用外挂、脚本等违规方式参与活动
2. **活动时间**：请在活动有效期内参与，过期无法补做
3. **任务验证**：系统会自动验证任务完成情况，请勿尝试作弊
4. **奖励领取**：请及时领取奖励，过期将视为自动放弃

### 违规处理

| 违规行为 | 处理方式 |
|----------|----------|
| 使用外挂或脚本 | 取消活动资格，清除违规数据，严重者永久封禁 |
| 恶意刷取奖励 | 取消活动资格，收回违规奖励 |
| 利用系统漏洞 | 取消活动资格，收回违规奖励，上报漏洞 |

### 活动变更

系统保留在必要时变更活动规则、奖励和时间的权利，变更前会提前通知。

## 技术说明（仅供开发者参考）

游戏活动系统架构包含：
- **活动触发器服务**：处理时间触发、用户行为触发和数据阈值触发
- **活动统计服务**：计算参与率、完成率、转化率、漏斗分析等
- **活动模板系统**：支持活动模板 CRUD 和版本管理
- **定时任务调度**：基于 cron 表达式的活动调度
- **WebSocket 实时推送**：活动状态变更和用户进度实时推送
