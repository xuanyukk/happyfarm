## 五、设计规则总结

### 获取途径分层原则

```
商店购买 ← 基础途径，所有道具均可购买
    ↓
收获掉落 ← 概率途径，作物等级越高→道具品质越高
    ↓
每日任务 ← 稳定途径，固定奖励+概率额外掉落
    ↓
成就奖励 ← 里程碑途径，一次性高品质道具
    ↓
玩家升级 ← 等级途径，里程碑道具包
    ↓
活动奖励 ← 限时途径，限定道具/高概率掉落
```

### 任务与成就边界规则

| 判定标准 | 每日任务 | 成就系统 |
|----------|---------|---------|
| 重复性 | ✅ 可无限次重复 | ❌ 一次性完成 |
| 数值上限 | ✅ 无硬上限 | ❌ 有明确上限 |
| 刷新周期 | 每日重置 | 永久性 |
| 奖励类型 | 货币+道具(少量) | 货币或道具(大量) |
| 示例 | 种植N次、出售N次 | 品质升级、地块解锁 |
