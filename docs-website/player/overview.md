# 功能特性

本文档介绍开心农场项目的主要功能特性。

## 核心功能

- 🎮 用户系统：注册、登录、JWT 认证、密码重置、双因素认证
- 🎮 会话管理：设备管理、强制下线、会话安全
- 🎮 农场管理：50 块固定地块、8 档品质覆盖、品质星级系统（5 个星级/品质）、地块可视化
- 🎮 作物系统：84 种作物（基础15种+经济16种+稀有38种+顶级15种）、完整的种植-收获-出售循环
- 🎮 [多货币体系](/features/multi-currency-system)：多货币架构（农场币+宝石），策略模式扩展，万/亿格式化显示，999亿上限，货币流水记录
- 🎮 商店系统：种子和道具购买、等级解锁、每日限购重置
- 🎮 背包系统：种子、作物、道具库存管理、批量出售、槽位上限 200（数据库可配置）
- 🎮 活动日志：完整的玩家操作记录、分类筛选、搜索功能
- 🎮 成就系统：82 条成就（种植/收获/商业/地块/品质/等级/世界/道具/签到/社交/活动/隐藏/连击/收藏）、成就进度展示、分类管理、统计信息、稀有度分级（普通/稀有/史诗/传奇/神话）
- 🎮 [游戏活动系统](/guide/game-events)：活动管理、任务配置、进度追踪、奖励发放、实时推送
- 🎮 体力值系统：基础上限 200，玩家等级≥300 解锁至上限 1000，自动恢复
- 🎮 道具系统：30 种道具（增产剂4种+加速剂4种+特殊效果6种+属性强化5种+限时皮肤3种+装饰2种+体力恢复3种）、yield_boost/speed_boost 范围 1.0~10.0，显示百分比+剩余时间
- 🎮 数据备份：自动定时备份、手动备份、队列管理
- 🎮 操作审计：完整的审计日志、权限变更追踪

## 扩展功能

- ⚙️ 角色权限管理（RBAC）：用户-角色-权限三级架构、功能级/操作级/数据级权限控制
- ⚙️ 游戏公告系统：富文本编辑、分类管理、定时发布、阅读状态追踪
- ⚙️ 游戏参数配置：参数分类、多数据类型支持、版本管理、审批流程、导入导出
- ⚙️ 批量操作功能：列表勾选、批量操作、Excel 导入导出、异步处理
- ⚙️ 实时预警推送：WebSocket 长连接、预警规则配置、定向推送、处理记录
- ⚙️ 配置热更新：配置缓存管理、批量编辑、导入导出、缓存状态监控
- ⚙️ 日志分析与可视化：日志查询、统计分析、可视化图表、实时监控

## 游戏活动系统

- 🎮 活动管理系统：活动创建、任务配置、进度追踪、奖励发放、统计监控
- 🎮 活动触发器系统：时间触发、用户行为触发、数据阈值触发
- 🎮 活动模板系统：模板CRUD、版本管理、变量配置、模板渲染
- 🎮 活动WebSocket服务：活动状态广播、用户进度实时推送、奖励领取通知
- 🎮 活动统计服务：参与率、完成率、转化率、漏斗分析
- 🎮 定时任务调度：cron表达式、固定时间调度、任务重试、失败处理

## 数据仓库与 BI 分析

- 📊 维度表：日期维度、玩家维度、作物维度
- 📊 事实表：日活跃玩家、每日交易、作物种植、每日收入
- 📊 聚合视图：DAU 统计、作物统计、收入统计、留存分析
- 📊 ETL 任务：自动数据抽取、转换、加载
- 📊 BI 分析 API：完整的数据分析接口

## 监控告警体系

- 🔔 服务器资源监控：CPU、内存、磁盘、网络使用率实时监控
- 🔔 告警分级策略：P0-P3 四级告警，响应时间明确
- 🔔 多渠道告警通知：控制台、邮件、企业微信、钉钉、短信
- 🔔 告警阈值动态调整：支持自定义告警阈值
- 🔔 告警闭环管理：告警触发、确认、处理、关闭完整流程

## 业务指标监控

- 📈 交易成功率监控：登录成功率、支付成功率、数据保存成功率
- 📈 用户活跃度监控：在线用户数、日活跃用户、峰值在线统计
- 📈 游戏业务指标：活跃玩家、作物收获、交易次数、商店访问量
- 📈 性能指标监控：API 响应时间、QPS、错误率
- 📈 自定义指标支持：支持注册和更新自定义业务指标
- 📈 多维度分析：综合指标视图、分类查看、历史分析
- 📈 趋势预测：基于历史数据的线性回归预测
