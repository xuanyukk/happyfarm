# 核心玩法

本文档介绍开心农场的核心游戏玩法。

## 游戏流程

1. **注册/登录**：访问注册页面创建账号，或登录进入游戏
2. **查看农场**：进入首页，查看 50 块地块状态
3. **购买种子**：访问商店，购买适合等级的种子
4. **种植作物**：点击空地选择种子，确认种植
5. **等待成熟**：查看生长进度条和剩余时间
6. **收获作物**：点击成熟地块或使用"一键收获"
7. **出售作物**：访问背包，出售作物获得农场币
8. **解锁地块**：使用农场币解锁新地块
9. **提升品质**：点击地块查看品质信息，消耗资源提升
10. **提升星级**：点击地块提升星级（★☆☆☆☆ ~ ★★★★★），星级越高产量/速度加成越大
11. **品质晋级**：普通品质 50 块地块全部五星后，可解锁铜品质升级
12. **查看成就**：点击首页"🏆 成就"，查看游戏进度
13. **管理队列**（可选）：访问队列管理，管理后台任务
14. **查看流水**：点击"📊 流水"，查看交易记录和统计

## 体力值系统

- 体力值基础上限：200
- 玩家等级达到 300 时，体力值上限解锁至 1000
- 体力值低于 20% 时显示红色警告
- 体力值自动恢复，显示恢复倒计时
- 种植作物消耗体力值，体力不足时无法种植

## 地块品质与星级

- 8 种品质：普通 → 铜 → 铁 → 金 → 翡翠 → 钻石 → 狱 → 无尽
- 每种品质含 5 个星级（★☆☆☆☆ ~ ★★★★★）
- 星级升级消耗农场币，星级越高收益越大
- 品质升级条件：当前品质所有地块达到五星后才能升级到下一品质

## 道具效果

- yield_boost（产量加成）：1.0~10.0，1.0=基础产量，10.0=10倍产量
- speed_boost（速度加成）：1.0~10.0，1.0=基础速度，10.0=10倍速度
- 道具使用后显示对应图标+剩余时间+效果百分比
